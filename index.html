Parallax Scrolling Effects

Parallax scrolling's been around for a while but it's one of those web design trends that just keeps on going. Parallax scrolling is when the website layout sees the background of the web page moving at a slower rate to the foreground, creating a 3D effect as you scroll. Used sparingly it can provide a nice, subtle element of depth that results in a distinctive and memorable website.
